# Politica de privacidad de Brainki

Ultima actualizacion: 18 de mayo de 2026

Esta Politica de privacidad explica como Brainki recopila, usa, almacena y protege la informacion relacionada con el uso de la app. Brainki es una app educativa y de bienestar familiar pensada para ayudar a niños y familias a organizar rutinas, practicar foco, realizar misiones pequeñas, usar herramientas de calma y seguir progreso de forma ludica.

Brainki no es una app medica, no diagnostica, no trata y no reemplaza atencion profesional medica, psicologica, terapeutica ni educativa.

## Informacion que puede usar Brainki

Brainki puede guardar o usar la siguiente informacion:

- Nombre o apodo del perfil.
- Avatar o emoji elegido.
- Puntos, niveles, racha, misiones completadas y progreso dentro de la app.
- Preferencias de configuracion, como tema visual, recordatorios y opciones sensoriales.
- Datos necesarios para iniciar sesion, como identificador de usuario, correo o nombre, cuando se usa Google Sign-In o Sign in with Apple.
- Estado de suscripcion Premium, prueba gratuita y compras, gestionados por Apple mediante App Store.
- Datos compartidos con el widget mediante App Group, como puntos, nivel, racha, nombre del perfil y progreso necesario para mostrar el widget.

## Como se usa la informacion

La informacion se usa para:

- Personalizar la experiencia de Brainki.
- Mostrar el nombre, avatar, progreso, puntos, niveles y recompensas.
- Recordar misiones completadas y configuraciones elegidas.
- Actualizar el widget de Brainki.
- Programar recordatorios locales cuando el usuario o adulto responsable los activa.
- Verificar si existe una suscripcion Premium activa mediante Apple.
- Permitir inicio de sesion con proveedores compatibles.

Brainki no vende datos personales.

## Niños y supervision adulta

Brainki esta pensada para usarse con supervision de una madre, padre, tutor o cuidador responsable. La app puede ser usada por niños, pero la configuracion de cuenta, suscripciones, compras, enlaces externos y decisiones importantes deben ser revisadas por un adulto.

No se debe ingresar informacion sensible innecesaria del niño, como direccion, escuela, ubicacion exacta, documentos de identidad, informacion medica detallada o cualquier dato que no sea necesario para usar la app.

## Servicios externos

Brainki puede usar servicios externos para funciones especificas:

- Google Sign-In: para iniciar sesion con una cuenta de Google.
- Sign in with Apple: para iniciar sesion con Apple cuando esta disponible.
- App Store y StoreKit: para gestionar suscripciones, prueba gratuita, compras, renovaciones y restauracion de compras.

Estos servicios pueden procesar datos de acuerdo con sus propias politicas de privacidad.

## Suscripciones y pagos

Las compras, prueba gratuita y suscripciones de Brainki se procesan mediante Apple. Brainki no recibe ni almacena datos completos de tarjetas de pago. El usuario puede administrar o cancelar suscripciones desde su cuenta de Apple o App Store.

## Notificaciones

Brainki puede solicitar permiso para enviar notificaciones locales. Estas notificaciones se usan para recordatorios de foco, agua, descanso, misiones o progreso. El usuario puede desactivarlas desde la app o desde Ajustes del dispositivo.

## Widget

El widget de Brainki puede leer informacion limitada guardada localmente en el App Group de la app, como puntos, nivel, racha, nombre del perfil y mision del dia. Esto permite mostrar el progreso en la pantalla de inicio.

## Almacenamiento y seguridad

Brainki guarda informacion principalmente en el dispositivo y, cuando corresponde, en el App Group compartido con el widget. Se toman medidas razonables para limitar el acceso a informacion innecesaria. Ningun sistema es completamente infalible, por lo que se recomienda no ingresar datos sensibles que no sean necesarios.
